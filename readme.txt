- A base de dados é a Wine Data Set disponível em https://archive.ics.uci.edu/ml/datasets/wine.

- Há 1 atributo indicador de classe (1, 2 e 3) e 13 atributos de características encontradas em 3 tipos de vinhos.

- Foram retirados 3 instâncias do data set, uma de cada classe, para realizar a predição no final do código (linhas 64-68).

- Lembrar de alterar o caminho do graphviz na linha 50.

- O método aplicado foi o de árvore de decisão. 

- A figura wine.png ilustra a árvore de decisão e mostra o critério escolhido pelo treinamento da rede, os valores de entropia (impureza de variável), o número de amostras que cumprem com o critério definido e a classe correspondente aquele critério.
